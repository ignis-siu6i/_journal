## 🎮 스팀의 게임 영상 저장 구조

 스팀은 게임 클립이나 리플레이 데이터를 다음과 같은 방식으로 저장합니다:

- **경로:** 보통

```
php-template

코드 복사

`C:\Program Files (x86)\Steam\userdata\<사용자ID>\760\remote\<게임ID>\videos`
```

 또는

```
makefile

코드 복사

`C:\Users\<사용자명>\Videos\Steam`
```
- **파일 형식:** `.webm`, `.mkv`, `.dem`, 또는 스팀이 사용하는 자체 캡처 포맷(`.steamclip` 등)일 수 있습니다.
 → 이 경우 단순히 복사해도 다른 플레이어나 장치에서 바로 재생되지 않을 수 있습니다.

## 💾 백업 또는 내보내기 절차

 영상이 단순한 MP4 파일이 아니라면, 아래 절차를 따라 백업하거나 다른 장치에서 볼 수 있습니다.

### 1. **Steam에서 직접 내보내기 (가장 간단한 방법)**

- 스팀 클라이언트 → `보기(View)` → `스크린샷` 또는 `비디오` 메뉴로 이동
- 원하는 클립 선택 후 “**파일 보기(View on Disk)**” 클릭
- 해당 폴더에서 **원본 영상 파일을 복사**해서 외장하드나 클라우드로 백업

### 2. **일반 영상 파일로 변환하기**

 만약 `.webm`이나 스팀 전용 포맷이라면, 변환을 통해 범용 MP4로 만드는 것이 좋습니다.

- **FFmpeg** (무료 변환 도구) 예시:

```
bash

코드 복사

`ffmpeg -i input.webm -c:v libx264 -preset slow -crf 22 output.mp4`
```
- 변환 후엔 어떤 기기에서도 재생 가능하며, NAS나 외장 SSD에 복사해도 문제 없습니다.

### 3. **자동 백업 설정하기**

- 스팀 영상 폴더를 **OneDrive / Google Drive / Dropbox** 동기화 폴더로 지정하면 자동 백업이 가능합니다.
- 또는 윈도우 “파일 기록” 기능을 이용해 해당 폴더를 백업 목록에 추가할 수도 있습니다.

## ✅ 정리

| 상황 | 권장 조치 |
| --- | --- |
| 단순 .mp4 또는 .webm 파일 | 바로 복사 및 백업 가능 |
| 스팀 전용 포맷(.steamclip 등) | 스팀에서 내보내기 or FFmpeg 변환 |
| 자동 백업 원할 때 | 클라우드 동기화 폴더 연결 또는 윈도우 파일 기록 |
